All fungi are edible. Some fungi are only edible once.
-Terry Pratchett

Welcome to A Shortcut to Mushrooms! This mushroom classifier was written in python using machine learning to predict whether a mushroom is edible or poisonous based on 15 attributes of the specimen. In order to achieve this binary task, we used data from the Mushroom Project by Dennis Wagner, Dominik Heider, and Georges Hattab at Philipps Universitat in Marburg, Germany. We trained various machine learning models on classification based on a simulated dataset derived from common European mushrooms. We found Random Forest provided the best results, followed by a Neural Network and K-Nearest Neighbors. After training the model, we created a new simulated dataset based on North American species. We then combined these datasets in order to train the model on both North American and European mushroom species and were able to get highly reliable predictions.

If you are a foraging enthusiast feel free to explore our datasets with our interactive visualizations or make a prediction for a specific specimen based on various attributes of the mushroom’s cap, stem, gills and ring.

WARNING: THIS IS SOLELY FOR EDUCATIONAL PURPOSES AND IS NOT TO BE USED TO ACTUALLY DETERMINE THE EDIBILITY OF A MUSHROOM!
